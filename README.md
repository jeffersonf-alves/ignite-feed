# Ignite Feed

Projetinho feito em um curso da Rocketseat para entender melhor alguns conceitos de react.js.

## Assuntos abordados no curso

 - Estados com UseState
 - UseEffect
 - Imutabilidade
 - Propriedades
 - Componentização 
 - JSX
 - Vite JS
 - Renderização de listas
 - Manipulação de Eventos
